# Professional E-Commerce Store

A modern, professional Shopify-style e-commerce platform built with Flask. Features a clean design, dark mode, professional navigation, and complete store management capabilities.

## ✨ Features

### Store Features
- **Professional Design**: Shopify-inspired layout with professional typography
- **Dark Mode**: Toggle between light and dark themes
- **Responsive**: Mobile-first design that works on all devices
- **Product Management**: Full CRUD operations for products and categories
- **Shopping Cart**: Session-based cart with real-time updates
- **Checkout**: PayPal and Cash payment options
- **Order Management**: Complete order tracking and management
- **Inventory**: Stock tracking with low-stock warnings

### Admin Features
- **Dashboard**: Professional admin panel with stats and quick actions
- **Product Management**: Add, edit, delete products with image uploads
- **Category Management**: Organize products into categories
- **Order Management**: View and manage customer orders
- **Settings**: Customize store appearance, contact info, and payment methods
- **User Management**: Secure admin authentication

### Technical Features
- **Supabase Storage**: Cloud image storage and CDN
- **PostgreSQL**: Production-ready database
- **Professional UI**: Bootstrap 5 with custom professional styling
- **Security**: CSRF protection, secure file uploads, input validation
- **Performance**: Optimized queries, lazy loading, efficient caching

## 🚀 Quick Start

### 1. Clone Repository
```bash
git clone <your-repo-url>
cd professional-ecommerce
```

### 2. Set Up Supabase
1. Go to [supabase.com](https://supabase.com) and create a new project
2. Go to **Storage** → Create bucket named `products` → Make it public
3. Go to **Settings** → **API** and copy:
   - Project URL
   - Service Role Key
4. Go to **Settings** → **Database** and copy the connection string

### 3. Deploy to Render
1. Push your code to GitHub
2. Go to [render.com](https://render.com) and create a new Web Service
3. Connect your GitHub repository
4. Render will automatically detect the `render.yaml` configuration
5. Add the following environment variables:

```
DATABASE_URL=your_supabase_postgresql_url
SUPABASE_URL=your_supabase_project_url
SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key
SUPABASE_BUCKET=products
SECRET_KEY=your_secret_key (auto-generated by Render)
```

### 4. Access Your Store
- **Store**: `https://your-app-name.onrender.com`
- **Admin**: `https://your-app-name.onrender.com/admin/login`
- **Default Login**: 
  - Email: `admin@example.com`
  - Password: `admin123`

## 🛠️ Local Development

### 1. Set Up Environment
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Environment Variables
Create `instance/config.env`:
```env
SECRET_KEY=your-secret-key-here
DATABASE_URL=sqlite:///app.db  # For local development
SUPABASE_URL=your_supabase_url
SUPABASE_SERVICE_ROLE_KEY=your_supabase_key
SUPABASE_BUCKET=products
DEFAULT_ADMIN_EMAIL=admin@example.com
DEFAULT_ADMIN_PASSWORD=admin123
```

### 3. Run Locally
```bash
python app.py
```
Visit `http://127.0.0.1:5000`
