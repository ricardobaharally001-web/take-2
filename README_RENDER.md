## ğŸš€ Quick Start

### 1. Clone Repository
```bash
git clone <your-repo-url>
cd professional-ecommerce
```

### 2. Set Up Supabase
1. Go to [supabase.com](https://supabase.com) and create a new project
2. Go to **Storage** â†’ Create bucket named `products` â†’ Make it public
3. Go to **Settings** â†’ **API** and copy:
   - Project URL
   - Service Role Key
4. Go to **Settings** â†’ **Database** and copy the connection string

### 3. Deploy to Render
1. Push your code to GitHub
2. Go to [render.com](https://render.com) and create a new Web Service
3. Connect your GitHub repository
4. Render will automatically detect the `render.yaml` configuration
5. Add the following environment variables:

```
DATABASE_URL=your_supabase_postgresql_url
SUPABASE_URL=your_supabase_project_url
SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key
SUPABASE_BUCKET=products
SECRET_KEY=your_secret_key (auto-generated by Render)
```

### 4. Access Your Store
- **Store**: `https://your-app-name.onrender.com`
- **Admin**: `https://your-app-name.onrender.com/admin/login`
- **Default Login**: 
  - Email: `admin@example.com`
  - Password: `admin123`
